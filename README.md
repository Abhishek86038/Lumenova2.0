# Lumenova-L2 Crowdfunding dApp

A real-time crowdfunding dApp built on Stellar Testnet using Soroban, React (Vite), and TailwindCSS.

## Features
- **Multi-Wallet Support:** Connect via Freighter, xBull, or Albedo using `@creit.tech/stellar-wallets-kit`.
- **Live Campaign Data:** Real-time sync of campaign progress and goal from the deployed Soroban contract.
- **Donate on Testnet:** Submit donations in XLM securely.
- **Real-Time Event Feed:** Subscribes to Soroban RPC events to show recent donations immediately.

## How to run locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the Vite development server.
4. Ensure you have Freighter or xBull installed on Testnet to interact with the dApp.
